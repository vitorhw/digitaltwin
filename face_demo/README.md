# Face Avatar Standalone 

Runs independently from DigitalTwin 

## Quickstart 

```bash
python -m venv .venv
.venv\Scripts\activate
## (for above for you use VScode and want to create virtual environment)

pip install -r face_demo/requirements.txt

# Start on port 8001(inside face_demo folder)
python server.py
```
You also can use some testing images in 'test_image' folder 
You are free to use your own testing images

Open: http://localhost:8001
